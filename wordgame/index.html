Battle-Hangman

rules:

1. Pick a word.
2. computer will randomly pick a word.
3. each player gets 50 points to start with.
4. you must type a single letter into the guess and an arbitrary number in wager. Then hit the button that appears to make the computer take a guess and wager.
5. game is over when one word is fully guessed.
6. player with most points at the end wins.
